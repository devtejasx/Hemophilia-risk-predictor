## Troubleshooting

**Q: Chatbot says "I'm not sure"**
- A: You can still ask - it will try OpenAI API if configured
- A: Or try rephrasing your question
- A: If about hemophilia, try more specific terms

**Q: No answer appears**
- A: Check internet connection
- A: Try refreshing the page
- A: Try another question first

**Q: Want longer/detailed answers?**
- A: Set up OpenAI API key (optional)
- A: Go to `.env` file and add: `OPENAI_API_KEY=your_key_here`
- A: Then the smart chatbot will give more detailed responses

## Tips for Best Results

1. **Be specific:** "What should I eat?" vs. "List healthy foods for hemophilia"
2. **Ask follow-ups:** Ask clarifying questions to get more detail
3. **Try variations:** Rephrase if answer is not what you expected
4. **Use keywords:** For hemophilia topics, use terms like "inhibitor", "mutation", "severity"
5. **Multiple questions OK:** Ask as many as you want!
