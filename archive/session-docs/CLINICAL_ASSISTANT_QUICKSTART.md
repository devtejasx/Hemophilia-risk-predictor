# Structured Clinical Assistant - Quick Start (5 Minutes)

## 🚀 Quick Launch

1. **Start the app:**
   ```bash
   streamlit run app.py
   ```

2. **Navigate to:** 🤖 Advanced Chatbot

3. **Load patient data (optional):**
   - Go to 👥 Patient Form
   - Fill in patient details
   - Click "Save & Process"
   - Return to Advanced Chatbot

## 🎯 The 4 Modes Explained

| Mode | Best For | Icon |
|------|----------|------|
| **Diagnosis Support** | Understanding symptoms, differential diagnosis | 🔍 |
| **Treatment Recommendation** | Optimizing doses, comparing options | 💊 |
| **Risk Explanation** | Understanding risk scores, inhibitor risk | ⚠️ |
| **Monitoring** | Planning check-up schedules, protocols | 📊 |

## 💬 How to Use

### Step 1: Choose Your Mode
Click one of the 4 buttons at the top:
- 🔍 Diagnosis Support
- 💊 Treatment Recommendation  
- ⚠️ Risk Explanation
- 📊 Monitoring Analysis

### Step 2: Ask Your Question
Type in the chat box:
```
"What does my mutation mean?"
OR
"Is my dose appropriate?"
OR
"Why is my risk score high?"
OR
"What monitoring do I need?"
```

### Step 3: Get AI Response
The AI responds with:
- Your patient context (if available)
- Mode-specific reasoning
- Evidence-based recommendations
- Disclaimer reminder

### Step 4: Follow Up (Optional)
- Ask clarifying questions
- Get more examples
- Export conversation

## ⚠️ Critical Reminder

👉 **AI suggestions are NOT medical advice**

Always:
- ✓ Consult your specialist
- ✓ Report adverse events
- ✓ Never delay treatment
- ✓ Use AI for decision prep

## 🎯 Example Questions by Mode

### 🔍 Diagnosis Support
```
"My patient has joint swelling. What could it be?"
"How do I interpret this high factor level?"
"What's the differential diagnosis?"
```

### 💊 Treatment Recommendation
```
"Is 2000 IU 3x/week right for my patient?"
"Should we switch to extended half-life?"
"How do I improve adherence?"
```

### ⚠️ Risk Explanation
```
"My risk is 45%. Why?"
"What factors protect me?"
"Is my inhibitor risk high?"
```

### 📊 Monitoring Analysis
```
"How often should I check inhibitors?"
"What monitoring is needed?"
"When should I increase monitoring?"
```

## 📊 Patient Context

If you loaded patient data, AI automatically includes:
- Name, Age, Gender
- Severity, Mutation
- Treatment (Dose, Product)
- Risk score
- Joint damage, bleeding history

**All personalized to your patient!**

## 💾 Export Conversation

In the 📝 History tab:
1. Click "📥 Export Conversation"
2. Download as .txt file
3. Share with team
4. Document discussion

## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| No response | Check internet, retry |
| Wrong mode | Click correct button |
| No patient data | Complete Patient Form first |
| Missing context | Ensure Save & Process clicked |
| Slow response | May take 5-15 seconds |
